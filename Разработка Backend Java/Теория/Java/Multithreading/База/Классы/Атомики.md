Классы обертки над примитивами, которые используются для работы в конкурентной среде
Для изменения используются проверка версий
Если версия после чтения и перед изменением не совпадают, то значение читается заново и к нему прибавляется число

Основные методы
- .getAndIncrement()
- .incrementAndGet()
