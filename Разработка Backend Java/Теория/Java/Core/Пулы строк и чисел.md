**Пул строк** это пространство в куче, где хранятся строки, создаваемые как литералы.
Такие строки можно сравнивать через оператор `==`.

Чтобы вытащить строку из пула, нужно вызвать у нее метод `String.intern()`

Пул чисел хранит значения целочисленного (*Integer*) в диапазон от -128 до 127 типа в одном месте памяти для повторного использования.
