Внутри лежит классический HashMap и двусвязный список.
При добавлении элемент добавляется в обе структуры
Есть разные стратегии вставки, по умолчанию добавляется в конец.

## Производительность

- **Вставка**: O(1) в среднем случае
    
- **Поиск**: O(1) в среднем случае
    
- **Удаление**: O(1) в среднем случае
    
- **Итерация**: O(n) — быстрее, чем HashMap (т.к. идет по связному списку, а не по бакетам)
