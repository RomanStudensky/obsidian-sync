При чтении ничего не блокируется 

- При вставке новых элементов происходит блокировка нужного бакета
- При вставке в пустой бакет используются атомарные операции CAS (Compare And Swap)
- При перестройке(***resize***) мыпы потоки, вставляющие данные, помогают переносить данные из старой таблицы в новую
