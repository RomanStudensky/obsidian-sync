В корне две ссылки на первый и последний эл
Размер лежит в поле *size*

При добавлении создается нода и крепится в конец

Основные операции и их сложность:

- **Добавление/Удаление в начале (addFirst, removeFirst):**
    - Сложность: **O(1)**.
    - Причина: Просто меняются указатели `head` и `next`.
- **Добавление/Удаление в конце (addLast, removeLast):**
    - Сложность: **O(1)** (в `java.util.LinkedList`, так как есть указатель `tail`).
- **Добавление/Удаление в середине (add(index, element), remove(index)):**
    - Сложность: **O(n)**.
    - Причина: Необходимо найти узел по индексу (обход O(n)), затем изменить ссылки (O(1)).
- **Получение элемента по индексу (get(index)):**
    - Сложность: **O(n)**.
    - Причина: Поиск элемента с заданного индекса требует последовательного обхода узлов от начала или от ближайшего конца списка, что в худшем случае занимает O(n/2) ~ O(n).
- **Итерация по всем элементам:**
    - Сложность: **O(n)**.
    - Причина: Каждый узел нужно посетить.
