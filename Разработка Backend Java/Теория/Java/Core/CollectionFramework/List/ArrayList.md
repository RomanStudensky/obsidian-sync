**Основные операции и их сложность:**

- **Доступ (get(index)**): **O(1)**. Прямой доступ по индексу в массиве.
- **Добавление в конец (add(element))**: В среднем **O(1)**. Если массив не переполнен. В худшем случае (при расширении) — **O(n)**.
- **Вставка в середину (add(index, element))**: **O(n)**. Требует сдвига всех последующих элементов.
- **Удаление (remove(index))**: **O(n)**. Требует сдвига элементов, следующих за удаленным, влево.
- **Проверка наличия (contains(element))**: **O(n)**. Требует линейного поиска элемента.
