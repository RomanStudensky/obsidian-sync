Сеть доставки содержимого. 
Выступает в роли стороннего слоя кэша для контента. Используется для хранения статического содержимого страницы сайтов, файлы, картинки и т.д.

![[Pasted image 20251021155910.png]]

Когда пользователь делает запрос на получение ресурса, его перенаправляет на ближайший сервер, который отдаст контент. Тем самым уменьшается время доставки и происходит распределение нагрузки по признаку наименьшего времени отклика.

---

**Как работает**:
- пользователь делает запрос с помощью URL;
- если в кэше сервера CDN нет данных, он запрашивает их из оригинального источника;
- источник отдает данные серверу CDN;
- CDN кэширует и отдает их пользователю;
- другой пользователь запрашивает данные;
- если TTL не истек, то CDN вернет пользователю данные, иначе отправит к оригиналу.
![[Pasted image 20251021160148.png]]
---

**Нюансы**:
- кешировать только **часто используемые данные**;
- подобрать **TTL**;
- продумать **доступность** на случай отказа CDN;
- продумать **механизм удаления данных** из CDN;
- продумать **версионирование** при необходимости.
